## Overview

이 workspace는 TAAC KDD Cup 2026 참가를 위한 코드와 실험 기록을 관리합니다.  

- Competition: [KDD Cup 2026 TAAC](https://algo.qq.com/)
- Focus: Large-scale Recommendation, CTR/CVR Prediction
- Key Topic: Sequence Modeling + Feature Interaction
- Stack: FuxiCTR 기반 모델 실험 및 확장
